# Contributing to Agent Autonomy Slider

Thank you for your interest in contributing! 🎉

## Ways to Contribute

### 🐛 Report Issues
- Check if the issue already exists
- Use a clear, descriptive title
- Include browser/device information
- Provide steps to reproduce

### 💡 Suggest Features
- Check existing feature requests first
- Describe the feature clearly
- Explain why it would be useful
- Provide examples if possible

### 🔧 Submit Code
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Make your changes
4. Test thoroughly
5. Commit with clear messages
6. Push to your fork
7. Open a Pull Request

## Development Setup

```bash
# Clone your fork
git clone https://github.com/your-username/agent-autonomy-slider.git

# Open in browser
cd agent-autonomy-slider
open index.html
```

No build process required - it's pure HTML/CSS/JavaScript!

## Code Style

- **HTML**: Semantic, accessible markup
- **CSS**: BEM naming convention preferred
- **JavaScript**: Clear variable names, comment complex logic
- **Formatting**: Consistent indentation (2 spaces)

## Pull Request Guidelines

- Keep PRs focused and small
- Update README if needed
- Test on multiple browsers
- Add screenshots for visual changes

## Questions?

Open an issue with the "question" label!
